Welcome to My Memory Game!

This is a game where you need to match pairs of tiles. Playing is very simple - you turn over one tile and then try to find a matching tile. You should be able to make a match (if you have perfect recall). The scoring system incorporates this principle:

Guide

The timer starts when you open a pair of cards (matched or unmatched).

Each time you make a successful match you score points.

If you have perfect recall, you score a bonus point.
